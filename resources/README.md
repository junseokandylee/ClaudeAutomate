## Required Sizes for Production

For production builds, the following PNG sizes should be generated by a designer:

- 16x16 (favicon)
- 32x32 (Windows taskbar)
- 48x48 (Windows shortcut)
- 64x64 (macOS Retina)
- 128x128 (macOS standard)
- 256x256 (Windows high DPI)
- 512x512 (macOS high DPI)

## Conversion Tools

To convert SVG to PNG in multiple sizes, you can use:

1. **Online Tools**:
   - https://cloudconvert.com/svg-to-png
   - https://convertio.co/svg-png/

2. **Command Line Tools**:
   ```bash
   # Using ImageMagick
   convert icon.svg -resize 16x16 icon-16x16.png
   convert icon.svg -resize 32x32 icon-32x32.png
   convert icon.svg -resize 48x48 icon-48x48.png
   convert icon.svg -resize 64x64 icon-64x64.png
   convert icon.svg -resize 128x128 icon-128x128.png
   convert icon.svg -resize 256x256 icon-256x256.png
   convert icon.svg -resize 512x512 icon-512x512.png
   ```

3. **Design Software**:
   - Figma, Sketch, or Adobe Illustrator
   - Export at multiple resolutions
